Methodology
  • ✓Real tokenizer. Counts via — not a length/4 heuristic.
  • ✓No injected constants. Only the actual payload each arm puts into context is counted; the shared prompt cancels.
  • ✓Input tokens only. No model call is made, so output tokens are not estimated — and the “Token Tax” is an input-token claim.
  • ✓Strawman excluded. Worst-case baselines are shown but kept out of the headline so the number can't be inflated.
